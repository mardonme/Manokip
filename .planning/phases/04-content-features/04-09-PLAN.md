This plan ships the public recipe-rendering surface. List page + detail page + 1 RSC card primitive + 1 client primitive (LocaleFallbackBanner — could be RSC if no client interaction needed, but Phase 3 D-05 banner is a client component because it shows a dismiss button per the spec).

Sitemap helper extension makes recipe URLs discoverable by search engines per locale. Phase 3 plan 03-08 shipped `buildLocaleSitemapEntries` — this plan adds 1 more entry-builder for recipes.

Closure status: Standard execute plan. No new test files (Wave 0 e2e stubs flip in 04-12 closure). Exit gate: tsc clean + visit /uz/recipes + /uz/recipes/<seed-slug> in browser, both render.

<threat_model>

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Public visitor browser ← /[locale]/recipes/[slug] | RSC-rendered HTML; no client ProseMirror bundle; Tiptap body HTML pre-escaped by static-renderer + injected via dangerouslySetInnerHTML inside prose wrapper |
| TechArticle JSON-LD ← public visitor | <script type="application/ld+json"> with \\u003c escape (defends against headline/excerpt containing < breaking out of script element) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-XSS-01 | XSS | Tiptap body rendered to HTML on public page | mitigate | Static-renderer locked extension allow-list + built-in escapeHTML/escapeHTMLAttribute; tested in 04-02. |
| T-04-XSS-02 | XSS | TechArticle JSON-LD <script> substring in headline/excerpt/mentions | mitigate | JSON.stringify(obj).replace(/</g, '\\u003c') wrapper at the script-tag emit site (Phase 3 D-09 carry-forward; structural pattern present in src/app/[locale]/products/[slug]/page.tsx:108). |
| T-04-INFO-01 | Information Disclosure | Public read serving draft recipes | mitigate | getRecipeBySlug + findPublishedRecipes both filter WHERE recipes.status='published'. Sitemap helper similarly filters via findPublishedRecipes. |
| T-04-INFO-02 | Information Disclosure | hreflang exposing missing-locale slug returning 404 | mitigate | buildAlternates({ slugByLocale }) omits null slots — only emits hreflang for locales where the translation exists (Phase 3 carry-forward). |
| T-04-TAMP-01 | Tampering | Locale param injection on /[locale]/recipes/[slug] | mitigate | next-intl middleware + layout-level hasLocale guard (Phase 3 carry-forward — src/app/[locale]/layout.tsx:52). |
| T-04-TAMP-02 | Tampering | slug param SQL injection in resolver | mitigate | Drizzle parameterized eq() — no string interpolation in SQL (Phase 3 carry-forward). |
